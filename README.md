
# TaskFlow

## Overview

TaskFlow is a mobile To-Do List application developed as part of a Mobile Application Development course.

The application helps users organize their daily activities by creating, managing, editing, and completing tasks through a modern and user-friendly interface.

---

## Features

- Splash Screen
- Login Screen
- Dashboard
- Add New Task
- Task Management
- Task Statistics
- Responsive UI
- Modern Material Design
- Smooth Navigation

---

## Technologies Used

- FlutterFlow
- Flutter
- Dart
- Material Design 3
- Figma (UI Design)
